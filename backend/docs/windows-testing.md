# Windows Testing

Manual Windows validation will include:

1. Install the application.
2. Enroll a device once.
3. Sign out and sign back in.
4. Confirm automatic launch after Windows login.
5. Confirm system tray visibility.
6. Confirm lock, unlock, suspend, and resume behavior.
7. Confirm screenshots pause while locked or idle.
8. Confirm the admin dashboard shows the employee as online during an active session.
9. Confirm screenshots appear in the admin dashboard and are not visible in the employee app.
10. Create a team from the admin dashboard, add the employee, and verify team filters for screenshots, timesheets, and reports.
